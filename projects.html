Order Management App

This project is a web application designed to streamline order management in food service businesses such as restaurants and cafes. The frontend is developed using HTML, CSS, Bootstrap, and JavaScript, ensuring a user-friendly interface that adapts seamlessly to various devices. On the backend, Python and Flask were utilized, offering a robust and efficient framework for handling requests, managing sessions, and executing core functionalities. The database layer is powered by SQLAlchemy, a versatile Object Relational Mapper (ORM) that simplifies database operations and ensures data integrity. The main goal of this project was to create a lightweight yet functional tool that addresses the specific needs of small to medium-sized food establishments. Key features include streamlined order tracking, intuitive navigation, and a scalable design. You can explore the code and more details on my GitHub: OrderFlow
